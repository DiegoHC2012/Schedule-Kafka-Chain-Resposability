Module Actions

Cataloga productos, crea órdenes, registra pagos y mueve estatus desde un solo bloque.

Desktop: dos columnas y dos renglones · Mobile: tarjetas apiladas · Órdenes usan catálogo

Create Product

Da de alta el producto en catálogo para que luego pueda usarse en órdenes.

Create Order

Selecciona uno o varios artículos del catálogo, define la cantidad y arma la orden desde aquí.

Cada fila representa un artículo de la orden. Puedes agregar uno o varios productos del catálogo.

Register Payment

Más compacto: selección de orden arriba y monto/saldo en una fila cuando hay espacio.

Se calcula con base en la orden seleccionada.

Update Order Status

Acción corta, con espacio visual para que no quede comprimida junto al formulario de pagos.

Si cambias a PAGADO se generará el flujo de envío automático según la lógica del módulo.
